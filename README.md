# Nuxt 3 Boilerplate

- Nuxt3
- Folders and aliases
- EsLint, Prettier
- WindiCSS
- SASS
- Pinia

## Run docker container (and install dependencies)

```bash
yarn up
```

## Stop docker container

```bash
yarn down
```

## Rerun docker container

```bash
yarn reboot
```

## Access project shell

```bash
yarn shell
```
